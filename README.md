## Install

```shell
npm install hanzi-to-pinyin
```

## Usage

```js
const convert = require('hanzi-to-pinyin')
convert('我的猫喜欢吃苹果')
.then(console.log)
// [ ['wǒ'], ['de','dī','dí','dì'], ['māo'], ['xǐ huan'], ['chī'], ['píng guǒ'] ]
```
